# EDev Shell Tool
A cross platform shell script tool.

## Technologies
* Script Languages: cmd, bash, Powershell, vbscript

## Features
* Test Suite:
  * Terminal Emulators:
    * ConEmu
	* MinTTY (Cygwin, Msys2)
  * Shells:
    * cmd
	* Powershell
	* bash
  * Development Environments:
    * mingwm
	* mingw-w64
	* Cygwin
	* Msys2
* CMD Shell:
  * Self elevated prompt.
* Powershell Shell:
  * Self elevated prompt.
