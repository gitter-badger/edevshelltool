# EDevBashTool

A multi platform shell tool

## Features
* Detect Terminal Emulators (ConEmu, MinTTY);
* Detect Shells (cmd, Powershell, bash);
* Elevated cmd prompt;
* Elevated Powershell prompt;

## Technologies
* bash
* batch
* powershell
* cmd
* mingw
* mingw-w64
* cygwin
* msys2
